# chainer-ebgan
A Chainer implementation of EBGAN.

# Requirements
- Python3  
- chainer1.16

# Usage
## gan_mnist.py
GAN with MNIST(use GPU #n).
```
python dcgan_mnist.py -g n
```

## ebgan_mnist.py
EBGAN with MNIST(use GPU #n).
```
python dcgan_mnist.py -g n
```
